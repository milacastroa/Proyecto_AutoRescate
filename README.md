# PROYECTO-1-ESTRUCTURAS-DE-DATOS
# Proyecto de gestión de servicios AutoRescate 24/7

Descripción
Sistema para gestionar servicios de asistencia vehicular en carretera, incluyendo:
- Gestión de solicitudes
- Asignación de técnicos y unidades
- Priorización de casos críticos
- Control de recursos

Estructuras usadas
- Cola (FIFO) → solicitudes normales
- Pila (LIFO) → kits y reversión de operaciones
- (Posible) Cola de prioridad → casos críticos

Contenido
- /src → Código fuente
- /docs → Documento PDF

Restricciones
No se usa Java Collections Framework
